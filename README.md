📈 Digital Agency CRM & Project Delivery Analytics

📌 Project Overview
This project provides an in-depth operational analysis of a Digital Marketing Agency's project delivery lifecycle. By analyzing a simulated CRM dataset of 2,500 web development projects (WordPress, Salla, Custom) over two years, this dashboard identifies critical operational bottlenecks and delivery inefficiencies.

🚀 Key Business Insights
The "Client Review" Bottleneck: Data clearly highlights that the 'Client Review' phase is the primary cause of project delays, with actual days significantly exceeding expected timelines.
Delay Attribution: The analysis reveals that "Client Feedback Pending" is the leading reason for missed deadlines, shifting the focus from internal team performance to client management strategies.
Platform Efficiency: Visualized the average lead time by platform, demonstrating that 'Salla' projects have a significantly faster time-to-market compared to 'WordPress' and 'Custom Development'.
Clean SaaS UI: Developed a premium, standalone HTML dashboard utilizing CSS Grid to mimic a modern SaaS analytics interface.

🛠️ Tech Stack
Language: Python 3.x
Data Manipulation: Pandas, NumPy
Data Visualization: Plotly (Clustered Bar, Horizontal Bar, Donut Charts)
UI/UX: Standalone HTML/CSS Grid Dashboard (Light SaaS UI)
